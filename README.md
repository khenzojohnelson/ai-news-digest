# ai-news-digest
AI Agents yang bisa menyaring berita harian paling penting (nasional dan internasional) dan menyajikannya ke pembaca dengan nuansa pembawaan seperti misalnya The 1984 by George Orwell dan menambahkan analisis mendalam. Mulai dari 5W+1H, teori-teori dibalik berita dan hubungannya yang dibahas perlahan, dampak dan urgensi, pesan moral yang reflektif, hingga analisis objektivitas sumber-sumber berita. AI akan melakukan analisis terhadap berbagai sumber berita (BBC, Guardian, dan sebagainya) dan menjelaskan arah narasi berita dari berbagai sumber itu. Hasil dikirim melalui discord dalam bentuk link google docs dan disimpan dalam folder google drive setiap harinya. Saya menggunakan Claude AI Sonnet-4 Model untuk merapikan dan menggenerate ulang kode pada project ini sehingga lebih terstruktur, efisien, dan logic workflownya lebih jelas untuk dimengerti. Dalam proses pembuatan, saya menggabungkan Discord Webhook URL, GROQ API (Chatgpt OSS 120b parameter model), NewsAPI, dan Google Cloud Console (services bot untuk menciptakan folder&file di Google Drive). Personalisasi AI dapat diubah sesuai keinginan sendiri.
