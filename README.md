# ai-news-digest
AI Agents yang bisa menyaring berita harian paling penting (nasional dan internasional) dan menyajikannya ke pembaca dengan nuansa pembawaan seperti cerita Novel karya George Orwell dan menambahkan analisis mendalam (5W+1H, teori-teori dibalik berita dan hubungannya yang dibahas perlahan, dampak dan urgensi, dan pesan moral yang reflektif). Hasil dikirim melalui discord dalam bentuk link google docs dan disimpan dalam folder google drive setiap harinya. Saya menggunakan Claude AI Sonnet-4 Model untuk merapikan dan menggenerate ulang kode pada project ini sehingga lebih terstruktur, efisien, dan logic workflownya lebih jelas untuk dimengerti. Dalam proses pembuatan, saya menggabungkan Discord Webhook URL, GROQ API (Chatgpt OSS 120b parameter model), NewsAPI, dan Google Cloud Console (services bot untuk menciptakan folder&file di Google Drive). Personalisasi AI dapat diubah sesuai keinginan sendiri.

Bagaimana project ini bermanfaat: Anda akan mendapatkan wawasan global dari berita terbaru yang populer, dan disajikan dengan analisis sangat lengkap dan konsolidasi antar konsep yang mendalam. Selain itu, anda juga bisa mendapatkan analisis tambahan dengan nuansa yang anda sukai tanpa perlu membuang beberapa menit hanya untuk membuka chatgpt, menulis prompt, menunggu respons, dan menyimpan drafnya ke tempat penyimpanan anda secara manual. Project ini cocok untuk orang yang tertarik mempelajari isu nyata penting secara mendalam dan mencintai efisiensi.
