# ai-news-digest
AI Agents yang bisa menyaring berita harian paling penting (nasional dan internasional) dan menyajikannya ke pembaca dengan nuansa pembawaan seperti cerita Novel karya George Orwell dan menambahkan analisis mendalam. Mulai dari 5W+1H, teori-teori dibalik berita dan hubungannya yang dibahas perlahan, dampak dan urgensi, pesan moral yang reflektif, hingga analisis objektivitas sumber-sumber berita. Tenang. Sangat kecil kemungkinannya bacaan anda untuk bias. Hal ini karena AI akan melakukan analisis terhadap berbagai sumber berita (BBC, Guardian, dan sebagainya) dan menjelaskan arah narasi berita dari berbagai sumber itu dan selalu mengingatkan anda untuk berhati-hati dengan informasi yang beredar. Hasil dikirim melalui discord dalam bentuk link google docs dan disimpan dalam folder google drive setiap harinya. Saya menggunakan Claude AI Sonnet-4 Model untuk merapikan dan menggenerate ulang kode pada project ini sehingga lebih terstruktur, efisien, dan logic workflownya lebih jelas untuk dimengerti. Dalam proses pembuatan, saya menggabungkan Discord Webhook URL, GROQ API (Chatgpt OSS 120b parameter model), NewsAPI, dan Google Cloud Console (services bot untuk menciptakan folder&file di Google Drive). Personalisasi AI dapat diubah sesuai keinginan sendiri.

Bagaimana project ini bermanfaat: Anda akan mendapatkan wawasan global dari berita terbaru yang populer, dan disajikan dengan analisis sangat lengkap dan konsolidasi antar konsep yang mendalam. Selain itu, anda juga bisa menyesuaikan pembawaan berita tanpa membuang beberapa menit hanya untuk membuka chatgpt, menulis prompt, menunggu respons, dan menyimpan drafnya ke tempat penyimpanan anda secara manual. Project ini cocok untuk orang yang tertarik mempelajari isu nyata penting secara mendalam dan mencintai efisiensi. Maksud saya, anda cukup menyiapkan segelas kopi anda, membuka discord, memencet link, dan menikmati bacaan anda. Semuanya akan tersimpan secara cloud. Merasa tidak tertarik dengan bacaan anda? Pencet ulang commit di github dan dalam waktu 30 detik, anda sudah mendapatkan apa yang anda inginkan.
